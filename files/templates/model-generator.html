    ℹ️ How It Works

    • Define Targets: Set your desired RTP, volatility, and hit frequency
    • Random Search: Generate 200 random reel configurations
    • Evaluate: Calculate theoretical metrics for each candidate
    • Optimize: Select the configuration closest to your targets
    • Export: Use the generated weights in your live game

    ⚙️ Technical Details

    • RTP: Mathematical expectation per unit wagered
    • Volatility (σ): Standard deviation of returns
    • Hit Frequency: P(any win) = P(triple) + P(pair)
    • Fitness Function: Weighted multi-objective optimization
    • Reel Slots: 9 total slots per reel, each symbol ≥1
